Java HTML DOM

Con el HTML DOM, JavaScript puede acceder y cambiar todos los elementos de un documento HTML.

El HTML DOM es un modelo objeto estándar y una interfaz de programación para HTML. Este define:

  • los elementos HTML como objetos;
  • las propiedades de todos los elementos HTML;
  • los métodos para acceder a los elementos HTML; y
  • los eventos para cualquier elemento HTML;

1 El HTML DOM (Documet Object Model)

Cuando cargamos una página web, el buscador crea un Document Object Model) de la página.

El modelo HTML DOM se construye como un árbol de objetos:

Con el modelo de objeto, JavaScript accede a todo lo necesario para crear un HTML dinámico. JavaScript puede

  • cambiar todos los elementos HTML de la página;
  • cambiar todos los atributos HTML de la página;
  • cambiar todos los estilos CSS de la página;
  • remover los elementos y atributos HTML existentes;
  • añadir nuevos elementos HTML y atributos;
  • "reaccionar" (react) a todos los eventos HTML existentes en la página;
  • crear nuevos eventos HTML en la página;

2 Métodos HTML DOM

Los métodos HTML DOM son acciones que puede realizar en elementos HTML. Las propiedades HTML DOM son valores de elementos HTML que puede reestablecer o cambiar.

2.1 La interfaz de programación DOM

Se puede acceder al DOM HTML con JavaScript (y con otros lenguajes de programación). En el DOM, todos los elementos HTML se definen como objetos. La interfaz de programación se compone de las propiedades y métodos de cada objeto.

Una propiedad es un valor que puedes obtener o definir (como cambiar el contenido de un elemento HTML).

Un método es una acción que puedes hacer (como añadir o remover un elemento HTML).

Ejemplo 1
<p id="demo"></p>

<script>
document.getElementById("demo").innerHTML = "Hello world!"
</script>

En este ejemplo, getElementById es un método y innerHTML es una propiedad.

El método getElementById

La manera más comun de acceder a un elemento HTML es usando el id del elemento. En el ejemplo, el método es usado para acceder al elemento cuyo id es "demo".

La propiedad innerHTML

La manera más fácil de acceder al contenido de un elemento es mediante la propiedad innerHTML. Esta propiedad es útil para obtener o reemplazar el contenido de elementos HTML.
